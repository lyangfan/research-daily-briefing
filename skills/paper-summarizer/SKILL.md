# 科研论文总结 Skill

这个 Skill 专门用于总结科研论文，突出研究问题、方法创新和关键结果。

## 使用场景

- 总结 arXiv、bioRxiv、medRxiv 等预印本平台的论文
- 生成中文摘要，突出核心贡献
- 提取关键信息用于早报系统

## 总结格式

### 中文总结模板

请按以下结构总结论文：

```
【研究问题】
简要描述论文要解决的核心问题或挑战

【方法创新】
- 主要方法或框架
- 技术创新点
- 与现有方法的区别

【关键结果】
- 核心实验结果
- 性能指标或改进幅度
- 方法有效性验证

【应用价值】
- 适用场景
- 实际意义
- 局限性（如有）
```

### 要求

- 200-300 字（中文）
- 简洁清晰，避免技术细节堆砌
- 突出创新点和贡献
- 使用专业但易懂的语言

## 输入格式

调用时请提供以下信息：

```json
{
  "title": "论文标题",
  "authors": ["作者1", "作者2"],
  "abstract": "论文摘要",
  "categories": ["分类1", "分类2"],
  "url": "论文链接",
  "platform": "平台名称"
}
```

## 示例

### 输入

```
标题: Multi-Agent Reinforcement Learning for Drug Discovery
摘要: We propose a novel multi-agent reinforcement learning framework for molecular optimization...
```

### 输出

```
【研究问题】
传统药物发现过程耗时长、成本高，如何利用 AI 加速新药研发是关键挑战。

【方法创新】
- 提出多智能体强化学习框架，将分子优化建模为协作游戏
- 每个智能体负责不同子任务（分子生成、性质预测、合成路径规划）
- 引入新颖的奖励塑形机制，平衡探索与利用

【关键结果】
- 在虚拟筛选任务中，发现活性分子数量是基线方法的 3 倍
- 合成路径预测准确率达到 85%，显著优于传统方法
- 训练时间减少 60%，且收敛更稳定

【应用价值】
该方法可广泛应用于早期药物筛选，大幅降低研发成本。当前局限是需要真实实验数据进一步验证。
```

## 注意事项

1. **准确性优先**：如果论文内容复杂，优先保证准确性而非简洁性
2. **避免术语堆砌**：解释技术术语，让非专业读者也能理解
3. **客观表述**：基于论文内容总结，不添加未验证的推断
4. **识别重点**：区分核心贡献和辅助内容
