【研究问题】
随着测序技术的快速发展，个体基因分型成本大幅降低，基因组预测和全基因组关联研究在动植物经济性状以及人类疾病的预测和候选基因定位方面得到广泛应用。贝叶斯回归模型因其对复杂性状未知遗传架构的灵活假设，在过去十年中发挥了重要作用。然而，现有软件工具在模型类型、方法数量、编程语言统一性等方面存在局限性，难以满足不同应用场景的需求。

【方法创新】
• 开发了 hibayes R 软件包，能够同时拟合个体水平、汇总水平和单步贝叶斯回归模型
• 集成了最丰富的贝叶斯方法（13种），包括 RR、A、B、Bpi、C、Cpi、L、BSLMM、R 等
• 首个在单个工具中同时支持三种模型类型的软件：个体水平、汇总水平和单步模型
• 采用实时模型残差策略构建参数的全条件分布，避免大矩阵运算的时间消耗
• 使用 R 和 C++ 混合编程，提供易学且灵活的用户体验

【关键结果】
• 功能对比显示，hibayes 是唯一同时支持三种贝叶斯模型的工具（brms、BGLR、JWAS、GCTB 仅支持 1-2 种）
• 支持的方法数量远超同类工具：13 种贝叶斯方法 vs BGLR（6种）、JWAS（7种）、GCTB（5种）
• 提供最全面的功能集：10 项核心功能（固定效应、随机效应、协变量、方差组分、标记效应、GEBV、PIP、WPPA 等）
• 版本号为 3.0.0，相比 brms（2.19.0）、BGLR（1.0.9）、JWAS（1.0.0）、GCTB（2.04.3）更为先进
• 在单步模型中，对基因分型个体的预测准确性高于个体水平贝叶斯模型，对非基因分型个体的预测准确性也显著高于基于谱系的 BLUP 模型

【应用价值】
hibayes 适用于人类、植物和动物的统计遗传学研究和实际应用。通过整合谱系、基因型和表型数据，充分利用大规模群体的表型观测值，即使在个体水平数据不可用的情况下（如人类研究中的隐私保护），也能基于公开可用的汇总统计数据进行有效分析。该工具统一了不同编程语言（R、Julia、C++）的功能，简化了输入文件格式和结果输出的差异处理，显著降低了用户在工具选择和使用上的时间成本。
